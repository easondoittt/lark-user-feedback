# 用研报告 · 画板模板与配色（飞书画板）

出板用 `beautiful-feishu-whiteboard` 的 **Reading Room**（克制 / Restrained 级）：奶油纸底 + 墨线框 + 暖色块。下面是已跑通、可直接复用的规则与骨架。配合主 SKILL 的 Step4。

## 0. 致命坑（先读）：飞书画板不渲染浅色文字

飞书把 `<whiteboard type="svg">` 转成画板节点时，**所有 `<text>` 的 fill 会被强制成深色**。实测白色 / 奶油 / `style=fill` / `named white` 全部失效。

- ❌ 错：`<rect fill="#0B0A09"/>` + `<text fill="#FFFFFF">P0</text>` → 渲染成黑底黑字，看不见。
- ✅ 对（小标签/药丸）：浅底 + 粗墨描边 + 墨字
  ```
  <rect ... rx="18" fill="#F6EBD8" stroke="#0B0A09" stroke-width="3"/>
  <text ... fill="#0B0A09">P0 · 最痛</text>
  ```
- ✅ 对（必须保留深色块做「最重」编码时）：深块**不放字**，标签放到块外的浅底上（墨字）；或在深块里叠一个浅色小 chip 再放墨字。
- 出板后**必下载缩略图肉眼验收**深底标签是否可读：`docs +media-download --type whiteboard --token <block_token>`。

## 1. 配色（Reading Room）

| 用途 | 色值 |
|---|---|
| 纸底 background | `#F6EBD8` |
| 墨线 / 墨字 ink | `#0B0A09` |
| 次要文字 | `#5E5E54` / `#8A8A80` |
| P0 最重 | `#0B0A09`（墨，填充块/数据条/气泡）|
| P0/P1 | `#DE916A`（陶土）|
| P1 | `#F1DAB1`（黄油）|
| P2 | `#D6C7CC`（藕粉）|
| P3 | `#F6EBD8` + 墨描边（仅描边）|

**优先级编码**：靠**填充深浅**（墨→陶土→黄油→藕粉→仅描边）+ **描边粗细**（P0 用 3~3.5px）+ **位置**。不要用高饱和多色。

## 2. 板式（对齐报告 6 段，每个 h1 至少一块板或表）

> **小样本铁律**：独立用户 ≤2 / 有效 <8 时，所有优先级标签写「**Pn 暂定**」，并在封面/矩阵角落标一行「⚠ 方向性信号，非统计结论」。别让漂亮排版给薄数据背书。

1. **结论速览（封面）** —— 5 行：左侧细色条(14px，按优先级填充) + 主题名 + 一句话 + 右侧优先级药丸 + **置信度小字**。P0 行左条墨色；P0 药丸「浅底 + 3px 墨边 + 墨字」。
2. **检索口径** —— 卡片/小表：关键词集 · 群 · 时间范围（检索起止 + 实际命中最早~最晚）· 纳入/剔除规则 · 四类计数（有效/背景/解法/运营卡片）。
3. **证据台账** —— **优先用 docx 原生表格**（不是画板）：ID · 日期 · 用户 · 原话 · 证据类型 · 跳转按钮。画板放不下长文+按钮。
4. **主题聚类**：①**优先级矩阵**（横轴频次、纵轴影响）气泡按优先级填充，P0 气泡墨填、标签 `Tn / Pn` 放气泡**下方**墨字，叠置信度；②**频次条形**：每主题一条，长度=聚合后反馈数，颜色=优先级，**可叠第二条细线表示独立用户数**，右侧优先级药丸。
5. **优先级评分** —— 表/板：行=主题，列=5 维(频次/影响面/阻断/替代成本/复现) 各填 高/中/低 + P 级 + 置信度。这是替代「拍脑袋优先级」的硬底。
6. **设计机会** —— Top 2~3 主题，每个一张卡：`用户任务 → 当前断点 → 设计机会 → 方案假设 → 验证方式` 横向五格流程。
7. **待验证项** —— 文字/列表板：缺什么数据 · 找谁/哪个源补 · 怎么验证。

## 3. 可复用骨架：带优先级药丸的「行」

```xml
<whiteboard type="svg"><svg xmlns="http://www.w3.org/2000/svg" width="1600" height="..." viewBox="0 0 1600 ...">
  <rect x="0" y="0" width="1600" height="..." fill="#F6EBD8"/>
  <rect x="28" y="28" width="1544" height="..." fill="none" stroke="#0B0A09" stroke-width="2.5"/>
  <text x="70" y="112" font-size="46" font-weight="700" fill="#0B0A09">标题</text>
  <line x1="70" y1="136" x2="1530" y2="136" stroke="#0B0A09" stroke-width="2"/>

  <!-- P0 行 -->
  <rect x="70" y="232" width="1460" height="78" fill="#F6EBD8" stroke="#0B0A09" stroke-width="2"/>
  <rect x="70" y="232" width="14" height="78" fill="#0B0A09"/>                 <!-- 左色条=墨=最重 -->
  <text x="112" y="268" font-size="26" font-weight="700" fill="#0B0A09">T1　主题名</text>
  <text x="112" y="296" font-size="18" fill="#5E5E54">一句话描述</text>
  <rect x="1330" y="252" width="180" height="38" rx="19" fill="#F6EBD8" stroke="#0B0A09" stroke-width="3"/>
  <text x="1420" y="277" text-anchor="middle" font-size="18" font-weight="700" fill="#0B0A09">P0 · 最痛</text>
  <!-- P1/P2/P3 行：把上面的 #0B0A09 左条/药丸描边换成 #DE916A / #F1DAB1 / #D6C7CC，药丸可浅底墨字 -->
</svg></whiteboard>
```
